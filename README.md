# Optimizador-de-rutas

Problema

En muchas ciudades, los usuarios pierden tiempo en desplazamientos por rutas congestionadas o poco eficientes. No siempre es fácil identificar la ruta más rápida entre dos puntos usando transporte público o privado.

El reto técnico consiste en desarrollar una aplicación que, usando datos geográficos de la ciudad, calcule rutas optimizadas para minimizar tiempo o distancia, ofreciendo una herramienta práctica y fácil de usar para mejorar la movilidad urbana.
